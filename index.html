Defruni

el proyecto Defruni se presenta como una iniciativa integral que utiliza tecnologías web modernas para abordar un problema ambiental importante al cambiar las percepciones y comportamientos hacia los desechos orgánicos, especialmente entre los jóvenes.

ORGANIZER EVENT

En conclusión, el proyecto Organizer Event se presenta como una solución tecnológica que utiliza herramientas modernas y eficientes para mejorar la comunicación y coordinación en la planificación de eventos, beneficiando tanto a los organizadores como al público interesado en participar.

Pet trace

se presenta como una aplicación web construida con tecnologías modernas y eficientes, con el objetivo de facilitar la búsqueda y reunificación de animales perdidos con sus dueños, involucrando activamente a la comunidad en este proceso.
